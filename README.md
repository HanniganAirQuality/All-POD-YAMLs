# All-POD-YAMLs
This repository collects all YAML files required for parsing all datafiles from monitors created by the HAQ Lab!

In the continued push for increased metadata in air quality monitoring, we want to work towards that standard as well. For us, that is going to start with these yamls. 

If you are unfamiliar with YAML, it is similar to JSON formatting, in that it has a hierarchial structure & is compatible with a lot of different coding languages. This makes it incredibly useful as a platform for metadata.
